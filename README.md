# Raffle #

Add entries to a raffle and draw random winners.
The most recent winner is displayed in red.

![Alt text](/lib/assets/raffler.png)

### Run these commands to try it out: ###

```
bundle
rails s

```

* This Ruby version 1.9.3 & Rails version 3.2.12
